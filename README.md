## 🧪 How to test by yourself

Copy the project 

```bash
  $ git clone https://github.com/guilhermehvsantos/Ignite-lab-react.git
```
    
Install the dependecies  

```bash
  $ npm install
```

Start the application 

```bash
  $ npm run start
```
